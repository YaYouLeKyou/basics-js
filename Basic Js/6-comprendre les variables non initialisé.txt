--------------Cour--------------

Lorsque les variables JavaScript sont déclarées, 
elles ont une valeur initiale non définie. 
Si vous effectuez une opération mathématique sur une variable non définie, 
votre résultat sera NaN, ce qui signifie "Pas un nombre". 
Si vous concaténez une chaîne avec une variable non définie, 
vous obtiendrez une chaîne littérale de "non défini".
