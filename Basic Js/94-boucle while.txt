---------------cour------------------

Vous pouvez exécuter le même code plusieurs fois en utilisant une boucle.

Le premier type de boucle que nous allons apprendre est appelé une whileboucle 
car elle s'exécute "tant que" une condition spécifiée est vraie et s'arrête une fois que cette condition n'est plus vraie.

var ourArray = [];
var i = 0;
while(i < 5) {
  ourArray.push(i);
  i++;
}

Dans l'exemple de code ci-dessus, la whileboucle s'exécutera 5 fois et ajoutera les nombres de 0 à 4 à ourArray.
