var myFirstName;
var myLastName;
myFirstName = "Yanes";
myLastName = "Hadiouche";

----------cour-----------------
Auparavant, nous avons utilisé le code

var myName = "votre nom";

"votre nom" est appelé une chaîne littérale. 
Il s'agit d'une chaîne car il s'agit d'une série de zéro ou plusieurs caractères entre guillemets simples ou doubles.

Créez deux nouvelles variables de chaîne: 
myFirstName et myLastName et attribuez-leur respectivement les valeurs de votre prénom et de votre nom.
