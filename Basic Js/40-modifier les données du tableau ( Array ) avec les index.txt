--------------cour---------------------


Contrairement aux chaînes, les entrées des tableaux sont modifiables et peuvent être modifiées librement.

Exemple

var ourArray = [50,40,30];
ourArray[0] = 15; // equals [15,40,30]

Remarque
Il ne devrait y avoir aucun espace entre le nom du tableau et les crochets, comme array [0]. Bien que JavaScript soit capable de traiter cela correctement, cela peut dérouter les autres programmeurs qui lisent votre code.

Modifiez les données stockées à l'index 0de myArraysur une valeur de 45.
